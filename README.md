# Professional Developer Portfolio

A clean, modern, and professional developer portfolio website built with React, TypeScript, and Framer Motion.

## Features

- **Modern Design**: Minimalist aesthetic with glassmorphism effects and smooth transitions.
- **Interactive Projects**: Detailed project cards with hover effects and a modal for deep dives.
- **Animations**: Silky-smooth animations using Framer Motion.
- **Responsive**: Fully responsive design for all devices.
- **Performance**: Optimized for speed and SEO.

## Tech Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Vanilla CSS (CSS Variables, Flexbox/Grid)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Locally**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Customization

- **Content**: Update `src/data/content.ts` with your own projects and skills.
- **Styles**: Modify `src/index.css` to change colors, fonts, and layout.
- **Images**: Replace placeholder images in `src/data/content.ts` with your own.

## Deployment (GitHub Pages)

1.  Update `vite.config.ts` with your repository name if needed (set `base: '/your-repo-name/'`).
2.  Push to GitHub.
3.  Enable GitHub Pages in repository settings (deploy from `gh-pages` branch or use a workflow).
